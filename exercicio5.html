Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <strong> </strong> que é uma tag semântica e recomendada no lugar da tag B (não semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <em> que também é uma tag semântica e recomendada no lugar da tag I (não semântica).

Texto marcado

Podemos criar também um texto marcado usando a tag mark.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno

. Lembrando que a tag para criar um texto grande, usando a tag <big> ficou em desuso, porém a tag <small> continua funcionando normalmente!

Texto deletado

Podemos marcar um texto como excluído para indiciar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois usando a tag semântica <ins>.

Texto sobrescrito

Para inserir coisas do tipo x20+3 usando a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O usando a tag <sub>
